                    7. INSTALLATION INSTRUCTIONS
================================================================================

--------------------------------------------------------------------------------
7.1 BACKEND INSTALLATION
--------------------------------------------------------------------------------

Step 1: Install Python 3.9+
  Download from: https://www.python.org/downloads/

Step 2: Create Virtual Environment
  cd backend
  python -m venv venv
  
  Windows: venv\Scripts\activate
  Linux/Mac: source venv/bin/activate

Step 3: Install Dependencies
  pip install -r requirements.txt

Step 4: Setup Environment Variables
  Copy .env.example to .env
  Edit .env with your configuration:
    NEO4J_URI=bolt://localhost:7687
    NEO4J_USER=neo4j
    NEO4J_PASSWORD=password
    GROQ_API_KEY=your_api_key_here

Step 5: Start Neo4j
  docker run -d --name neo4j \
    -p 7474:7474 -p 7687:7687 \
    -e NEO4J_AUTH=neo4j/password \
    neo4j:5.14

Step 6: Run Backend
  python main.py
  
  Server starts at: http://localhost:8000
  API docs at: http://localhost:8000/docs

--------------------------------------------------------------------------------
7.2 FRONTEND INSTALLATION
--------------------------------------------------------------------------------

Step 1: Install Node.js 16+
  Download from: https://nodejs.org/

Step 2: Install Dependencies
  cd frontend
  npm install

Step 3: Start Development Server
  npm start
  
  Server starts at: http://localhost:3000

Step 4: Build for Production
  npm run build
  
  Output: frontend/build/

--------------------------------------------------------------------------------
7.3 FULL STACK INSTALLATION (WINDOWS)
--------------------------------------------------------------------------------

Automated Script: start_all.bat
  1. Starts Neo4j (Docker)
  2. Starts Backend (Python)
  3. Starts Frontend (Node.js)

Manual Steps:
  1. Install Python 3.9+
  2. Install Node.js 16+
  3. Install Docker Desktop
  4. Run: start_all.bat

--------------------------------------------------------------------------------
7.4 TROUBLESHOOTING
--------------------------------------------------------------------------------

Issue: Neo4j connection failed
  Solution: Check Docker is running
  Command: docker ps

Issue: Module not found
  Solution: Reinstall dependencies
  Backend: pip install -r requirements.txt
  Frontend: npm install

Issue: Port already in use
  Solution: Change port in configuration
  Backend: Edit main.py (port 8000)
  Frontend: Edit package.json (port 3000)

Issue: CORS errors
  Solution: Check CORS configuration in main.py
  Ensure allow_origins includes frontend URL
